## 📖 About The Project

**TechInovasi Stack** is an integrated IT infrastructure solution designed for technology companies. This solution automates the installation and configuration of critical services that traditionally require manual procedures.

### 🎯 Key Features

- ✅ **Full Automation** - One command to run all services
- ✅ **Security First** - FTPS (TLS) + SSH key-based authentication
- ✅ **Persistent Data** - Docker volumes for all critical data
- ✅ **Cross-Platform** - Access from Windows, Linux, Mac
